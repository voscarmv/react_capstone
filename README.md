# Cat-alog

> A search engine for cat breeds online.

## Built With

- React
- Redux
- HTML
- VS Code

## How to Install

- Clone it with `git clone` 
- `cd` to the root directory.
- Run `npm install`
- Turn on the server with `npm start`
- Visit `http://localhost:3000`
